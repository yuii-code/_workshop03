http method เหมือน action อย่างนึง Ex.
http method        action
    get             find
    post            insert / create
    put             update ทั้งหมด
    patch           update บางค่าที่ต้องการ update ไม่จำเป็นต้อง update ทั้งหมด
    delete          delete
